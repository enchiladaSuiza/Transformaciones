Configuración

Python

Este proyecto se realizará en el leguaje de programación Python, un lenguaje de programación de alto nivel usado ampliamente en el desarrollo de software. Existen muchas formas de instalar Python, ya que existen paquetes de software que lo incluyen junto con otras herramientas de desarrollo. El lector puede elegir la forma de instalación que desee, si es que no tiene Python instalado en su equipo.

Se recomienda instalarlo desde la página oficial, ya que es la manera más fácil y minimalista de empezar a programar. Para esto se necesita visitar https://www.python.org/, y descargar la versión más reciente bajo la sección de Downloads. Después únicamente se necesita seguir las instrucciones del instalador.

Tras la instalación, también se recomienda añadir el directorio de instalación (donde se encuentra python.exe) al Path del sistema. Esto se puede conseguir presionando la tecla Windows, escribir "variables de entorno" y presionar Enter para abrir el panel de control de propiedades sistema. Desde aquí se pueden editar las variables de entorno dando click en el botón correspondiente.

En la ventana que se abre, ahora se busca la variable Path bajo las variables del sistema y se da doble click.

Ahora se necesita dar click en el botón New para añadir una nueva variable, y copiar la ubicación del directorio de la carpeta de Python. Esto nos permitirá correr Python desde la terminal y desde cualquier ubicación del sistema de archivos.

Instalación de librerías

Para este proyecto se usa PyQt, una extensa y robusta librería de interfaz de usuario, basada en la librería Qt para C++. Para su instalación se requiere correr el siguinte comando desde la terminal.

python -m pip install pyqt6

Se recomienda instalar PyQt 6 ya que es la versión más reciente y la que está en desarrollo.
