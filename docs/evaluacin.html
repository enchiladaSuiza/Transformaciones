Evaluación

Esta sección contiene una serie de ejercicios que permitirán evaluar el conocimiento obtenido mediante este objeto de aprendizaje. El lector deberá contestar cada uno de ellos, sin buscar la respuesta dentro del objeto de aprendizaje o en algún otro medio (a excepción del programa realizado en Python).

La evaluación se divide en tres partes. El lector deberá realizar cada una, una sola vez, y guardar el puntaje obtenido. Al finalizar la evaluación se mostrará cómo cotejar los resultados.

Parte 1

Te serán mostradas diez preguntas y deberás de elegir la respuesta correcta de entre cuatro respuestas para cada pregunta. Tienes cinco minutos para resolver cada una.
